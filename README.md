# pynb2latex

This is a super simple utility that converts a python notebook like file to a typst file.

The main goals of this script is to:

- Take a python script where """""" string converted to visible output in the typst document.
- Render outputs to png's and embed them in the document.
- Take any sypmy output convert it to typst and embed that in the document.

tylax
